### Installation

1. Run `docker-compose up -d`
2. Run `docker-compose exec tg_bot python init.py`
3. Run `docker-compose exec backend python init.py`
4. Open [TgBot](https://t.me/EasyDocsDeepTech_bot) or [Website](localhost)
5. Enjoy!


## Technologies

#### Backend

- Flask
- TelegramBotApi
- Docker
- Docker-compose
- PostgreSQL
- Swagger

#### Frontend

- JavaScript
- HTML
- CSS
